# Second Semester Course Overview

Welcome to the second semester. This semester builds upon the foundational knowledge from the first semester and introduces more specialized and advanced topics in computing and engineering. Below is an overview of the courses you will be taking.

## Courses for S2

### 1. Higher Mathematics 2 (ICE0122)

This course is a continuation of the mathematical concepts introduced in the first semester. It will likely delve deeper into advanced calculus, linear algebra, differential equations, and their applications in solving complex engineering and computing problems. Expect to build a strong analytical foundation for future technical subjects.

### 2. Electronics for Computing 1 (ICE0123)

This course serves as an introduction to the fundamental electronic principles and components that underpin modern computing systems. Key topics will include digital logic, circuit analysis, semiconductors, and the basic building blocks of computer hardware. You will gain a practical understanding of how electronic circuits process information.

### 3. Interactive Web Development (ICE0121)

This course focuses on the client-side technologies required to build dynamic and responsive websites. You will learn to create engaging user experiences using core web technologies like HTML, CSS, and JavaScript. The curriculum may also introduce you to popular frontend libraries or frameworks to build modern web applications.

### 4. Operating Systems (ICE0121)

A core course in computer science, this module explores the principles and design of operating systems. You will learn how an OS manages system resources such as the CPU, memory, and storage. Topics include process management, concurrency, memory allocation, file systems, and system security.
