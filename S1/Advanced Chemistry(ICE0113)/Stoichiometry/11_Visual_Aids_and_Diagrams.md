# Visual Aids and Diagrams for Stoichiometry

## Stoichiometry Conversion Flowchart

```
┌───────────┐     ┌───────────┐     ┌───────────┐     ┌───────────┐
│  Mass of  │     │  Moles of │     │  Moles of │     │  Mass of  │
│ Substance A├────►Substance A├────►Substance B├────►Substance B│
└───────────┘     └───────────┘     └───────────┘     └───────────┘
      │                 │                 │                 │
      │                 │                 │                 │
      ▼                 │                 │                 ▼
┌───────────┐           │                 │           ┌───────────┐
│ ÷ Molar   │           │                 │           │ × Molar   │
│  Mass A   │           │                 │           │  Mass B   │
└───────────┘           │                 │           └───────────┘
                        │                 │
                        ▼                 ▼
                  ┌─────────────────────────┐
                  │ × Stoichiometric Ratio  │
                  │  from Balanced Equation │
                  │     (Mole Ratio B:A)    │
                  └─────────────────────────┘
```

## Mole Relationships in a Balanced Equation

For the reaction: 2H₂ + O₂ → 2H₂O

```
        2 molecules         1 molecule         2 molecules
        of hydrogen         of oxygen          of water
            │                  │                  │
            ▼                  ▼                  ▼
        ┌─────────┐        ┌─────────┐        ┌─────────┐
        │   2H₂   │    +   │   O₂    │   →    │  2H₂O   │
        └─────────┘        └─────────┘        └─────────┘
            │                  │                  │
            ▼                  ▼                  ▼
        2 moles of         1 mole of          2 moles of
        hydrogen           oxygen             water
        (4.032g)           (32.00g)           (36.04g)
```

## Mass-to-Mass Conversion Example

For the reaction: 2Al + 3Cl₂ → 2AlCl₃

```
┌───────────────────┐
│ 5.40g aluminum    │
└─────────┬─────────┘
          │
          │ ÷ 26.98 g/mol
          ▼
┌───────────────────┐
│ 0.200 mol aluminum│
└─────────┬─────────┘
          │
          │ × (2 mol AlCl₃/2 mol Al)
          ▼
┌───────────────────┐
│ 0.200 mol AlCl₃   │
└─────────┬─────────┘
          │
          │ × 133.33 g/mol
          ▼
┌───────────────────┐
│ 26.7g AlCl₃       │
└───────────────────┘
```
